### 用你的话说说什么是数据库？  
是计算机中众多储存数据的程序中一种、  
但是 数据库这个程序为了解决以下的问题  

* 解决超大的数据量的问题
* 要提供非常多人，并且用户体验好
* 要在计算机出现问题不会丢失
* 各种复杂的存储需求
